*Инструкция по  запуску Игры Битва Героев*

Способ 1: Запуск через PyCharm
Открытие проекта:

Открой PyCharm.
Перейди в меню File -> Open и выбери папку с проектом.

Запуск скрипта:
В панели проекта слева найди файл heroes_fight.py.
Щелкни правой кнопкой мыши по файлу и выбери Run 'heroes_fight'.

Ввод данных:
В консоли внизу появится запрос на ввод имени героя. Введи имя и нажми Enter.
Игра начнется, следуй инструкциям в консоли.

Способ 2: Запуск через командную строку

Открытие командной строки:
Открой командную строку (cmd) или терминал.

Навигация к проекту:
Перейди в папку с проектом с помощью команды cd. Например: cd путь/к/папке/с/проектом
Выполни команду для запуска скрипта:python heroes_fight.py

Ввод данных:
Введи имя героя, следуя запросу в консоли, и игра начнется.

Способ 3.Запуск через Replit.com

Регистрация и вход:
Перейдите на Replit.com.
Зарегистрируйтесь или войдите в свою учетную запись.

Создание нового проекта:
Нажмите на кнопку + Create или New repl.
Выберите шаблон Python и дайте название вашему проекту, например, heroes_fight.
Нажмите Create Repl.

Вставка кода:
В открывшемся редакторе найдите файл main.py.

Скопируйте и вставьте в этот файл код игры "Битва героев" из  файла heroes_fight.py

Запуск игры:
Нажмите на зеленую кнопку Run в верхней части экрана.
Внизу появится консоль, в которой будет запущена игра.
Введите имя вашего героя, следуя запросу в консоли, и игра начнется.
